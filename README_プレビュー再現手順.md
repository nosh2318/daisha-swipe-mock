# 代車・試乗.com スワイプモック

## 内容

- `index.html` : モバイル向けスワイプUIと事業者向けPC画面の実装本体
- `assets/` : 車両外観・内装画像一式
- `代車試乗COM_UIUX設計書.html` : サービス、画面、デザイン、画像基準の設計書

## プレビュー方法

1. このフォルダを任意の場所へ配置する
2. `index.html` をブラウザで開く
3. 画面幅が979px以下では顧客向けモバイルUI、980px以上では事業者向けPC UIが表示される

外部ビルド、npm、サーバー、データベースは不要です。HTML・CSS・JavaScriptを1ファイルにまとめているため、画像フォルダとの相対位置を維持すればそのまま再現できます。

## 動作確認ポイント

- 3項目の簡易ヒアリング
- 代車／暮らし試乗の切り替え
- 配送エリア入力
- 好き／今回は違うのスワイプ
- 前の車へ戻る
- カード上の「内装を見る」「装備を見る」
- 内装ギャラリーと装備一覧
- 好きな車から予約確認へ進む画面
- PC幅でのサービス概要、紹介案件、車両、顧客体験タブ

## 車両画像ルール

- 外観画像は車体の70%以上が見えること
- 前後端、ルーフ、タイヤが確認できること
- 外観カードは `object-fit: contain` を使用し、縦長画面への過度な拡大や切り抜きをしないこと
- 内装は運転席、後席、3列目、荷室を分けて登録すること

## 現在の位置づけ

本ファイルはUI/UX検証用モックです。空車、配送、予約処理は仮データであり、正式版では車両マスター、配車表、予約データとの連携が必要です。
